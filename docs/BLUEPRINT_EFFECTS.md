# Blueprint effects taxonomy (Route B)

Single source of truth for Architect output and Engineer implementation. Every key supports `"none"`.

## buttonStyle

- `none` — default solid or no special treatment
- `solid` — solid fill
- `gradient` — gradient on button background
- `outline` — outline only
- `glass` — glassmorphism
- `softGlow` — soft glow
- `bordered` — bordered
- `minimal` — minimal style
- `pill` — pill shape
- `neon` — neon-style

## frameBackdrop

- `none` — no backdrop effect
- `glow` — glow behind central frame
- `pulse` — pulsating glow
- `softGlow` — soft glow
- `particles` — particles
- `gradientRing` — gradient ring
- `shimmer` — shimmer
- `halo` — halo
- `subtleShadow` — subtle shadow

## entranceEffect

- `none` — no entrance animation
- `confetti` — confetti on first load
- `particles` — particles
- `fade` — fade in
- `scaleIn` — scale in
- `subtleDrift` — subtle drift
- `blurIn` — blur in
- `stagger` — staggered
- `floatUp` — float up

## cardContainer

- `none` — default
- `glass` — glassmorphism
- `softBorder` — soft border
- `elevated` — elevated
- `minimal` — minimal
- `gradientBorder` — gradient border

## typographyTreatment

- `none` — default
- `subtleShadow` — subtle text shadow
- `gradientText` — gradient text
- `letterSpacing` — letter spacing
- `allCaps` — all caps
- `serif` — serif
- `rounded` — rounded
